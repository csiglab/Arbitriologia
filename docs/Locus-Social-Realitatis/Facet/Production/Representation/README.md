# Production Representation

> A production system can be represented through multiple complementary abstractions.

> Each representation captures different structural aspects of how inputs, processes, and technologies combine to generate outputs.

| **Representation Aspect** | **Description**                                                                                                                                                                                                                                            |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Production Function**   | A **mathematical mapping** that relates input quantities to maximum attainable output under a given technology. Captures the *technological frontier*, substitution patterns, scale properties, and efficiency structure.                                  |
| **Production Network**    | A **graph- or matrix-based representation** of how multiple production stages, firms, or sectors are interconnected through inputâ€“output flows. Captures interdependencies, propagation of shocks, bottlenecks, supply chains, and system-level structure. |
